![Difficulty](https://img.shields.io/badge/Difficulty-Medium-orange)

Horcrux is a custom-built vulnerable machine designed to simulate a realistic multi-stage web exploitation scenario. It is intended for learners to practice enumeration, web application exploitation, and Linux privilege escalation in a controlled environment.

---

## Overview

This lab mirrors common real-world attack patterns found in misconfigured web applications and Linux systems. The machine requires methodical enumeration and exploitation across multiple layers without providing direct hints.

---

## Objective

Obtain root access on the target machine.

---

## Technical Stack

- Ubuntu Server
- Apache Web Server
- MySQL Database
- PHP
- WordPress

---

## 🛠 Skills Practiced

- Web application enumeration  
- File upload exploitation  
- Remote code execution (RCE)  
- Reverse shell handling  
- Database interaction (MySQL CLI)  
- Credential manipulation  
- Linux privilege escalation  
- Cron job analysis and abuse
